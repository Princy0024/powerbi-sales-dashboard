# Power BI Sales Dashboard

Interactive Power BI dashboard analyzing sales trends, shipments, revenue, and country performance.

## Dashboard Preview

![Dashboard Screenshot](Power_BI.jpeg)

## Overview

This interactive Power BI dashboard analyzes:

- Sales trends over time
- Country-wise shipment performance
- Product-level analysis
- Revenue movement using waterfall charts

## Tools Used

- Power BI
- Microsoft Excel

## Key Insights

- Australia recorded the highest shipment volume
- Revenue fluctuated across months
- Dark chocolate products performed strongly
